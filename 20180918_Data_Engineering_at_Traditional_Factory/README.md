## async 資料工程 await Serial port

邁入工業 4.0 中如何從設備裡擷取訊息進行數據分析並產生有用的回饋是重要任務。工廠設備裡使用 Serial port 串接硬體資料是常見的介面，本次 talk 將分享將會介紹為何會使用 Python asyncio 函式庫來建構橋接 Serial port 的資料工程以及過程中遭遇的問題及解決方法。
